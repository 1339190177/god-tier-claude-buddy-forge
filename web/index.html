📖 如何使用生成的 userID？
  1. 复制上方找到的 userID
  2. 打开文件 ~/.claude.json (Mac/Linux) 或 %USERPROFILE%\.claude.json (Windows)
  3. 删除 companion 字段（如果存在）
  4. 删除旧的 userID 字段（如果存在）
  5. 添加新的 userID： 
    {
  "userID": "复制的userID粘贴在这里"
}
  6. 重启 Claude Code
  7. 运行命令 /buddy 领取你的宠物！
